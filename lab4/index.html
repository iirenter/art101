Challenge

To create pseudocode for a everyday task and computer task. Then turn the computer task pseudo code into actual javascript comments.

Problems

We had no problems except for a little trouble coming up with tasks.

Results

We have the three lists here.

List 1 everyday task

Task: How to do Laundry

1.Have clothes ready in a pile or bag, Whites and colors separate

2.Place them into the washing machine

3.Add detergent

4.Set the settings

5.Turn on the washing machine

6.Wait until washing turns off

7.After washing machine is off, transfer clothes to the dryer

8.Add dryer sheets

9.Set the settings

10.Turn on dryer

11.Wait until dryer turns off

12.Take clothes from dryer and put them away

List 2 Computer task

Task: Create a basic calculator

1.Ask the user to input a number

2.Take in user input

3.Ask the user to input an operation

4.Take in user input

5.Ask user to input a second number

6.Take in user input

7.Calculate the result

8.Output/print the result
